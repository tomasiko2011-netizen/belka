# 🐿️ Белка — Казахстанская карточная игра онлайн

Онлайн-версия популярной карточной игры «Белка» с мультиплеером, текстовым чатом и видеозвонками.

## ✨ Возможности

- 🃏 **Полная игровая логика Белки** — постоянные козыри (Валеты), козырные масти, подсчёт глаз
- 👥 **Мультиплеер** — до 4 игроков в реальном времени через Pusher
- 💬 **Текстовый чат** — общение между игроками во время игры
- 📹 **Видеочат** — WebRTC видео и звук с миниокном справа
- 🛡️ **Модерация** — хост может мутить и банить игроков
- 📊 **Система очков** — глаза (шестёрки), подсчёт очков, яйца

## 🚀 Запуск

### 1. Установка зависимостей

```bash
npm install
```

### 2. Настройка базы данных (Turso)

```bash
# Установите Turso CLI
curl -sSfL https://get.tur.so/install.sh | bash

# Войдите в аккаунт
turso auth login

# Создайте базу данных
turso db create belka-game

# Получите URL
turso db show belka-game --url

# Создайте токен
turso db tokens create belka-game
```

### 3. Настройка Pusher

1. Зарегистрируйтесь на [pusher.com](https://pusher.com)
2. Создайте новое приложение
3. В настройках включите **Presence Channels**
4. Скопируйте ключи

### 4. Переменные окружения

```bash
cp .env.example .env.local
# Заполните все значения в .env.local
```

### 5. Инициализация БД

```bash
# Скопируйте SQL из lib/db.ts и выполните в Turso:
turso db shell belka-game < schema.sql
```

### 6. Запуск

```bash
npm run dev
```

## 🃏 Правила игры

**Белка** — командная карточная игра для 4 игроков (2 команды).

### Состав колоды
- 36 карт (с семёрками) или 32 (без семёрок)
- Команды: сидящие напротив за столом

### Козыри
- **Постоянные**: Валеты всегда козырь: ♣В > ♠В > ♥В > ♦В
- **Козырная масть**: определяется в каждом коне тем, у кого ♣В

### Ценность карт
| Карта | Очки |
|-------|------|
| Туз | 11 |
| 10 | 10 |
| Король | 4 |
| Дама | 3 |
| Валет | 2 |
| 7, 8, 9 | 0 |

Всего: **120 очков**

### Ход игры
1. Каждому раздаётся по 8 карт (или 7)
2. Первый ход — игрок слева от раздающего
3. Нужно класть карту той же масти, иначе — козырь или слив
4. Туз нельзя сбрасывать на масть, в которую ещё не ходили

### Подсчёт очков (Глаза)
- Команда с >60 очков открывает **глаза** (очки партии)
- Обычно открывается 2 глаза
- Если козырь был у проигравших и они не набрали «спас» (30+) — 3 глаза
- **Яйца**: по 60 у каждой → глаза удваиваются следующий кон
- **Победа**: 12 глаз!

## 📹 Видеочат

Видеочат работает на WebRTC с Pusher в роли сигнального сервера:
1. Нажмите 📹 чтобы включить камеру
2. Нажмите 🎙 чтобы включить микрофон
3. Миниокна игроков появятся в правой панели
4. Хост может мутить (🔇) и банить (🚫) игроков

## 🛡️ Модерация

Хост комнаты может:
- **Мутить** игроков в чате (они не смогут отправлять сообщения)
- **Банить** игроков (они будут исключены из комнаты)
- Нажмите ⚙ на аватаре игрока для управления
